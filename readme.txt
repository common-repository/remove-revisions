This plugin is able to automatically remove revisions older than n days.

It is available in:
<ul>
 <li>German</li>
 <li>English</li>
</ul>

== Installation ==

1. Upload the folder `remove-revisions` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. See the Options General -> Writing for the settings
